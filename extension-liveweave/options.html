🔒 Pair Foreman LiveWeave

In Foreman (the desktop app), open Pair browser extension — it shows a short code. Type it below. The code never leaves your machine; this extension proves it over a loopback challenge/response and pairs as the liveweave harness.

Empty means operator-token only. Use any only when every connected harness should be able to drive the canvas.

Foreman must be running locally (default http://127.0.0.1:54321). Everything stays on this device.
